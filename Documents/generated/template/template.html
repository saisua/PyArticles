2. Code

We can decorate any function an it will be converted to a text representation of it
summarize_users: Summarizes user information by calculating the average age of users.
Inputs: List[Int] ages, Integer min_age

Output: Dict[Str, Union[Int, Float, Nonetype]]

0 | total_age = 0
1 | num_users = 0
2 |
3 | for age in ages:
4 | if(age >= min_age):
5 | total_age += age
6 | num_users += 1
7 |
8 | average_age = total_age / num_users if num_users > 0 else None
9 |
10 | return {
11 | 'num_users': num_users,
12 | 'average_age': average_age
13 | }
Example 1: summarize_users([1, 2, 3, 4, 5, 6], 3): {'num_users': 4, 'average_age': 4.5} Example 2: summarize_users([1, 2, 3, 4, 5, 6], 7): {'num_users': 0, 'average_age': None}
